# XML Archive Processor

**A professional utility to convert single or multiple XML files into a single JSON file — fast, reliable, and easy to use.**

---

## Key Features

- ✅ Process a **single XML file** or an **entire folder** with multiple XML files  
- ✅ Choose between **CLI (Command-Line Interface)** or **GUI (Graphical Interface)** modes  
- ✅ **Auto-closing GUI** for batch processing workflows — run and forget  
- ✅ **No external dependencies** — works with Python standard library only  
- ✅ Designed for professionals, researchers, and developers who need fast XML → JSON conversion  

---
